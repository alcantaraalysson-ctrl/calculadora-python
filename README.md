# calculadora-python

Pequena calculadora em Python, projeto de prática.

## Funções

- `add(a, b)` — soma dois números
- `subtract(a, b)` — subtrai dois números
- `multiply(a, b)` — multiplica dois números
- `divide(a, b)` — divide dois números (lança erro se `b` for zero)
